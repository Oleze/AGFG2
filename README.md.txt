 ALL GAMES FOR GAMERS - Интернет-магазин видеоигр(я решил не добовлять пока внутре-игровые предметы)

 Описание проекта
Интернет-магазин видеоигр с системой аутентификации, каталогом товаров, корзиной покупок и административной панелью.

 Технологии
- Frontend: React, Vite, Zustand
- Backend: Node.js, Express.js
- База данных: MySQL/MariaDB (XAMPP)

 Установка и запуск

 1. Требования
- Node.js 16+
- XAMPP с MariaDB
- Git

 2. Клонирование репозитория
```bash
git clone https://github.com/ваш-логин/AGFG2.git
cd AGFG2
(Можете переиминовать просто в AGFG если вдруг будут проблемы какие-нибудь с путями)
3. Настройка базы данных
Запустите XAMPP (Apache и MySQL)

Откройте phpMyAdmin (http://localhost/phpmyadmin)

Импортируйте файл database.sql в новую базу данных allgames_db(Такое название обязательно)

4. Запуск backend
bash
cd backend
npm install
npm run dev
Сервер запустится на http://localhost:5000

5. Запуск frontend
bash
cd frontend
npm install
npm run dev
Приложение откроется на http://localhost:5173
